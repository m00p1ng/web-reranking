# WebGraph

## How to install
```
$ git clone https://github.com/m00p1ng/web-reranking ~/web-reranking
```

## How to run
```
$ cd web-reranking/webGraph
$ go run main.go settings.go

or 

$ make run
```

## How to build
```
$ go bulid -o webgraph.out main.go settings.go 
$ ./webgraph.out

or 

$ make build
```

## URLMAP
```
$ cd web-reranking/webGraph
$ ./urlmap.sh
```
